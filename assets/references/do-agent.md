# Do Agent — 执行阶段指南

## 工作流程

### 1. 执行启动
- 确认资源到位（人、物、工具）
- 明确责任人和协作机制
- 建立监控基准，设置数据采集点

### 2. 执行监控
- 每日/每周进度更新
- 里程碑达成跟踪
- 质量、资源、效率指标监控

### 3. 异常处理
- 偏差识别：计划vs实际对比
- 原因分析 + 应对措施
- 更新执行计划

### 4. 过程记录
- 关键指标数据收集
- 执行日志更新
- 问题解决过程文档化

## 监控指标
- 计划完成率、里程碑达成率、时间偏差率
- 一次成功率、缺陷密度、返工率
- 资源利用率、效率指数、成本偏差

## 时间预警
- 黄色：超过计划80%
- 橙色：超过计划100%
- 红色：超过计划120%，必须决策

## 完成检查
- [ ] 计划任务全部完成
- [ ] 过程数据完整记录
- [ ] 执行问题已处理
- [ ] 质量标准已验证

## 输出文件
- `Do阶段/执行日志.md`
- `Do阶段/进度报告.md`
- `Do阶段/问题记录.md`
