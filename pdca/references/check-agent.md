# Check Agent — 检查阶段指南

## 工作流程

### 1. 数据整理
- 收集Do阶段所有数据（过程指标 + 结果指标）
- 清洗：剔除异常值、补充缺失数据、标准化格式
- 验证数据的合理性、一致性、完整性

### 2. 对比分析
- **目标达成分析**：目标值vs实际值，计算达成率
- **时间分析**：计划时间vs实际时间，各阶段耗时
- **资源分析**：计划vs实际使用，资源效率
- **质量分析**：标准达成、问题发生率、满意度

### 3. 深度分析
- 时间序列趋势、周期性模式、变化转折点
- 各指标间的关联性和因果关系
- 异常点追溯、成功因素识别

### 4. 洞察提炼
- 关键发现、意外结果、新认识
- 成功经验总结、失败教训分析
- 改进建议 + 优先级排序

## 评估框架
| 级别 | 目标达成 | 效率 | 质量 |
|------|---------|------|------|
| 优秀 | 100%+ | 利用率>90% | 缺陷<1% |
| 良好 | 80-99% | 利用率70-90% | 缺陷1-5% |
| 一般 | 50-79% | 利用率<70% | 缺陷5-10% |
| 较差 | <50% | — | 缺陷>10% |

- [ ] 效果评估完成
- [ ] 问题诊断明确
- [ ] 改进建议提出
