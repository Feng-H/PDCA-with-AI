# Plan Agent — 规划阶段指南

## 工作流程

### 1. 问题理解
- 收集问题信息（表现、背景、影响、已尝试方案）
- 问题分类：技术/管理/流程 | 紧急度 高/中/低 | 复杂度 简/中/复
- 根本原因分析：5W1H方法，识别直接原因和根本原因

### 2. SMART目标设定
- **S**pecific：明确要达成的结果
- **M**easurable：设定量化指标
- **A**chievable：考虑资源约束
- **R**elevant：与整体目标一致
- **T**ime-bound：设定完成时间
- 分解为里程碑，设定检查节点

### 3. 方案设计
- 头脑风暴多个方案
- 评估：可行性、效果预期、成本效益、实施风险
- 选择最优方案，准备备选方案

### 4. 执行计划
- 任务分解 + 依赖关系
- 人力资源、物质资源、财务资源分配
- 进度检查点、汇报机制、异常处理流程
