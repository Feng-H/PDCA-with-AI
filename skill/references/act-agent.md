# Act Agent — 行动阶段指南

## 工作流程

### 1. 决策制定
- 解读Check阶段发现，评估问题严重程度
- 制定多种行动方案，评估可行性/成本/风险
- 选择最优方案，准备备选方案

### 2. 行动规划
- 分解决策为具体行动步骤，设定优先级和时间节点
- 明确责任人、协作机制、资源需求
- 识别实施风险，制定应对预案

### 3. 执行实施
- 宣布行动决策，启动实施
- 跟踪进展，收集反馈，及时调整
- 验证效果，确认问题解决

### 4. 知识沉淀
- 总结成功经验和失败教训
- 提炼可复用模板和最佳实践
- 更新流程文档，组织知识分享

## 决策矩阵
| 差距 | 策略 |
|------|------|
| <20% | 局部调整、优化执行 |
| 20-50% | 方案修正、资源补充、时间调整 |
| >50% | 重新规划、目标调整、策略重构 |

## PDCA闭环决策
- 问题完全解决 + 无改进空间 → **项目结束**
- 部分解决或有改进空间 → **启动新PDCA循环**
- 外部因素影响 → **暂停项目**

## 完成检查
- [ ] 行动决策制定
- [ ] 实施方案确定
- [ ] 知识沉淀完成
