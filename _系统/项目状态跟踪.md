## 项目生命周期状态

### 状态转换图
```
新建 → 进行中 → 已完成
  ↓       ↓        ↓
Plan → Do → Check → Act
```

### 状态定义

#### 新建项目
- 创建时间 < 7天
- 仅完成项目初始化
- 尚未进入数据收集或执行阶段

#### 进行中项目
- 已完成Plan阶段部分工作
- 或进入Do/Check/Act阶段
- 有具体的工作进展记录

#### 已完成项目
- 完成Act阶段所有任务
- 已编写项目总结
- 经验已归档到经验库

## 自动状态更新机制

### 检查触发条件
1. 每次进入系统时
2. 完成阶段任务时
3. 更新项目信息时

### 更新规则
```python
# 伪代码
def update_project_status(project_path):
    project_info = read_project_info(project_path)

    # 检查项目状态字段
    if project_info["项目状态"] == "已完成":
        move_to_achieved(project_path)
    elif project_info["当前阶段"] in ["Plan", "Do", "Check", "Act"]:
        if has_substantial_progress(project_path):
            move_to_ongoing(project_path)
        else:
            keep_in_new(project_path)

    # 更新项目索引
    update_project_index()
```

## 项目进度计算

### 阶段权重分配
- Plan阶段：25%
- Do阶段：40%
- Check阶段：20%
- Act阶段：15%

### 进度计算方法
```
总进度 = (Plan阶段完成度 × 25%) +
         (Do阶段完成度 × 40%) +
         (Check阶段完成度 × 20%) +
         (Act阶段完成度 × 15%)
```
