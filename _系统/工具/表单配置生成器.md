# 表单配置生成器

## 概述

**用途**: 根据项目选择的MECE框架，动态生成数据采集表单的字段配置。

**核心价值**:
- 标准化数据采集流程
- 自动匹配框架维度与数据字段
- 支持多场景数据采集需求

---

## 字段类型映射

| 数据类型 | Bitable type | 说明 |
|---------|-------------|------|
| 数字 | 2 | 数值类型，支持小数 |
| 单选 | 3 | 预定义选项，单选 |
| 多选 | 4 | 预定义选项，多选 |
| 是/否 | 3 | 特殊单选：是/否 |
| 文本 | 1 | 字符串类型 |
| 日期 | 5 | 日期时间类型 |

**Bitable字段创建API参数**:
```json
{
  "field_type": 2,
  "property": {
    "formatter": "0.00",
    "default_value": "0"
  }
}
```

## 表单创建流程

### 1. 项目创建阶段

```
项目创建 → 选择MECE框架 → 获取表单配置 → 创建Bitable字段
```

**步骤**:
1. 用户创建项目时选择框架（TREND/4M1E/自定义）
2. PlanAgent根据框架获取对应的字段配置
3. 在Bitable的`data_records`表中创建字段
4. 生成表单分享链接

### 2. 字段创建示例

```javascript
// 创建训练时长字段
await bitable.appTableField.create({
  table_id: "data_records",
  field_type: 2,
  property: {
    formatter: "0",
    default_value: "0"
  }
})
```

### 3. 表单分享

使用Bitable的表单视图功能生成分享链接：
```javascript
// 获取表单视图
const formView = await bitable.appTableViews.create({
  table_id: "data_records",
  view_type: "form"
})

// 生成分享链接
const shareUrl = `https://example.feishu.cn/base/${app_token}/form?view=${formView.view_id}`
```

## 数据写入流程

### 用户填写表单

```
用户填写表单 → 数据写入data_records表 → Cron检测新数据 → 仪表板更新
```

### 数据流

1. **表单提交**: 用户通过表单填写数据
2. **数据写入**: 数据自动写入`data_records`表
3. **自动巡检**: Cron任务检测到新数据
4. **仪表板更新**: 触发仪表板重新计算和渲染

### Cron检测逻辑

```javascript
// 检测新数据
const lastCheck = await getLastCheckTimestamp()
const newRecords = await bitable.appTableRecord.list({
  table_id: "data_records",
  filter: {
    conditions: [{
      field_name: "created_time",
      operator: "isGreater",
      value: [lastCheck]
    }]
  }
})

if (newRecords.length > 0) {
  await updateDashboard(project_id)
}
```

## 扩展自定义框架

### 自定义字段定义

当使用自定义MECE框架时，需要提供：

```json
{
  "framework_name": "自定义框架名称",
  "dimensions": [
    {
      "name": "维度名称",
      "fields": [
        {
          "name": "数据项名称",
          "type": "数字|单选|多选|是/否|文本|日期",
          "frequency": "采集频率",
          "unit": "单位"
        }
      ]
    }
  ]
}
```

### 字段验证规则

- 所有字段必须有明确的`name`和`type`
- 数值类型必须指定`unit`
- 单选/多选类型必须提供`options`
- 所有字段必须指定`frequency`

## 与其他组件集成

### 与仪表板生成器集成

表单字段与仪表板维度直接对应：
- 表单字段 → 仪表板数据源
- 采集频率 → 刷新频率建议
- 数据类型 → 可视化图表类型

### 与任务自动生成器集成

基于采集频率自动生成数据采集任务：
- 每日字段 → 创建每日提醒任务
- 每周字段 → 创建周报任务
- 每月字段 → 创建月度汇总任务

---

## 注意事项

1. **字段不可变性**: 表单创建后，字段类型不可修改
2. **数据完整性**: 必填字段需在创建时标记
3. **并发控制**: Bitable不支持并发写入，需串行调用
4. **权限管理**: 表单分享链接需设置适当的访问权限
