# 制造与质量项目模板库

## 设备OEE提升类项目模板

### 标准项目结构
- 问题调研：设备现状数据收集
- 原因分析：六大损失分析
- 方案制定：改善措施规划
- 效果确认：OEE提升验证

### 常见问题类型
- 设备故障率高
- 换型时间长
- 速度损失大
- 不良品率高

### 时间规划建议
- Plan阶段：7-15天
- Do阶段：30-60天
- Check阶段：7天
- Act阶段：15天

- FMEA分析

## 过程能力指数提升类项目模板

### Cpk改善步骤
1. 过程稳定性确认
2. 数据收集分析
3. 中心值偏移改善
4. 散差减少改善
5. 效果验证

### 目标设定
- Cpk < 1.33：需要改善
- Cpk ≥ 1.33：基本满足
- Cpk ≥ 1.67：良好
- Cpk ≥ 2.0：优秀

## 通用项目要素

### 团队组成建议
- 项目负责人：部门主管
- 核心成员：工程师、技术员
- 支持部门：质量、生产、设备、采购
- 顾问：相关领域专家

### 数据收集要求
- 基线数据：改善前3个月
- 目标数据：明确量化目标
- 过程数据：每日/每周跟踪
- 验证数据：改善后1个月

### 风险管理
- 技术风险：方案可行性
- 资源风险：人力、预算
- 时间风险：进度控制
- 质量风险：方案效果

## 项目成功要素

1. **高层支持**：获得管理层承诺
2. **数据驱动**：基于数据做决策
3. **全员参与**：调动团队积极性
4. **持续改进**：不满足于现状
5. **标准化**：经验固化和传承
